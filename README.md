# FitTrack – Kalori Takip Uygulaması

FitTrack, kullanıcıların günlük yedikleri yemekleri girerek kalori ve makro besin (protein, karbonhidrat, yağ) değerlerini takip etmesini sağlayan bir mobil uygulama projesidir.

## Özellikler
- Günlük hedef belirleme (örneğin 2500 kcal)
- Yiyecek ekleme (isim, kalori, makro değerler)
- Günlük toplam özet ekranı
- Basit ve sade arayüz tasarımı

## Teknolojiler
- Kotlin  
- Android Studio  
- Room Database  
- Material Design
